# Web服务器
## v1 
- 返回服务器URL的路径部分
  - 例如，如果请求的URL是http://localhost:8080/hello，响应则是URL.Path="/hello"
- main函数将一个处理函数和以/开头的URL链接在一起，代表所有的URL使用这个函数处理，然后启动服务器监听进入8000断口处的请求
- 一个请求由一个http.Request类型的结构体表示，它包含很多关联的域，其中一个是请求的URL，使用fmt.Printf格式化，然后作为响应发送出去
